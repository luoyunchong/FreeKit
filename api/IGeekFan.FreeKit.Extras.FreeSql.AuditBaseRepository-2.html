    Class AuditBaseRepository<TEntity, TKey>

    审计仓储：实现如果实体类 继承了ICreateAuditEntity 则自动增加创建时间/人信息 继承了IUpdateAuditEntity，更新时，修改更新时间/人 继承了IDeleteAuditEntity，删除时，自动改成软删除。仅注入此仓储或继承此仓储的实现才能实现如上功能。
